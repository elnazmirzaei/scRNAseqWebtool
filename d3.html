The diffusion map shows relationships between individual cell transcriptomes each represented by a sphere. The default radius can be changed to reflect the relative expression level of Genes or Regulons, with Linear Scaling (r=value*w) and Exponential Scaling (r=valuew). Colors represent stage of organismal development from which cells were isolated, or cluster association determined with Non-negative matrix factorization (NMF). The graph can be rotated by dragging.

Note:These data are the results of a collaboration, that has been submitted for publication, between the Spike Lab (Huntsman Cancer Institute) and Wahl Lab (The Salk Institute). Single cell suspensions were obtained from mammary epithelia at the given developmental stages and processed by DropSeq (10x Genomics) to obtain sequencing libraries. Expression values represent "expected counts" per gene from STAR --> RSEM, followed by normalization to adjust for sequencing depth.
Scaling can be altered for visualization using the buttons above. Positional information was generated using Diffusion Maps in the R destiny package at Bioconductor.org from Haghverdi et al 2015. This visualization has been plotted using Plotly package in JavaScripts.
